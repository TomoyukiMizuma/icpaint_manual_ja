    コードブロックの例¶

    markdown_extensions: pymdownx.highlight を有効にする。 コードブロック内の色はcssで指定する。

    def bubble_sort(items):
    for i in range(len(items)):
        for j in range(len(items) - 1 - i):
            if items[j] > items[j + 1]:
                items[j], items[j + 1] = items[j + 1], items[j]

    コードブロックに linenums="1" を入れると行数付きになる。 hl_lines="2 4" で指定する行数をハイライトにする。

    python linenums="1" hl_lines="2 4" def bubble_sort(items): for i in range(len(items)): for j in range(len(items) - 1 - i): if items[j] > items[j + 1]: items[j], items[j + 1] = items[j + 1], items[j]

    コードのインライン表示は gem install hoge のように ` を使う。

参考画像ウィンドウで画像を開くことでキャンバスとは別に画像を表示することができます。 <br />
イラストを制作する際の参考として使うことができます。

__"ツール選択"__ ボタンをクリックすることで参考画像を自由に動かすことができます。 <br />
ツール選択ウィンドウでも参考画像用のツールを選択することができます。
